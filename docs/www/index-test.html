Passcrow Test Server

This is a public server for testing passcrow.

You can enable it in your passcrow policy.rc by adding the following line: 

server: @SERVER_KINDS@ via @SERVER_NAME@

Please be warned that the operators make no guarantees about security or uptime, and the server is configured to delete any data that is over @KEEP_DAYS@ days old, no matter what expiration policy has been requested (or offered) by the software.

Note: If you are not just testing Passcrow, and actually want to be able to recover data, consider using passcrow.mailpile.is instead.
